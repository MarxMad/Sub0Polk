## üé® Design Guidelines

### Color Scheme
- **Primary**: Purple/Blue gradient (Web3 aesthetic)
- **Secondary**: Green (for checkmarks, success states)
- **Accent**: Orange/Yellow (for highlights, CTAs)
- **Background**: Dark mode or light gradient

### Typography
- **Headings**: Bold, sans-serif (Montserrat, Inter, or similar)
- **Body**: Clean, readable (Open Sans, Roboto)
- **Code**: Monospace (Fira Code, JetBrains Mono)

### Visual Elements
- **Icons**: Use emoji for quick visuals (‚úÖ ‚ùå üí∞ üöÄ etc.)
- **Logos**: Arkiv, Hyperbridge, Base, Ethereum, USDC
- **Screenshots**: DotGo UI, contract on Basescan, Arkiv queries

### Layout
- **Consistent margins**: 10% padding on all sides
- **Hierarchy**: Large titles, medium body, small footnotes
- **Whitespace**: Don't overcrowd slides
- **Alignment**: Left-align text, center-align titles
